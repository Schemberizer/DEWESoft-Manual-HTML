Initial Setup:

Information for the initial setup of the DEWESoft system

Powering Up the DEWESoft System:

  • Plug the 2-pin Lemo power cable into the "in" on the back of the system (Red)

  • Plug the micro-USB cable from the DEWESoft into the laptop (Green)

Opening the DEWESoft X Software:

  • Make sure the DEWESoft system is plugged in

  • Double click the DEWESoft X3 icon (Red)

  • If the DEWESoft system is not plugged in at first, plug it in and a screen will pop up. Press the button to retry (Green)






Analog Channel Setup:

Creating new analog channel setups and preparing the new setup for a test

Creating a New Channel Setup

  • Select the folder you wish to create the new setup in (Red)

  • Click the "New Setup" button (Blue)

  • Choose what analog channels will be used and name them (Green)

  • Change the sample rate of all channels (Orange)

  • Add all used channels to "Main Group" (Pink)

  • Set the range for the channel (Cyan)

Individual Channel Setup

  • Click the "setup" button for the channel you want to change (Red)

  • Change the measurement type, range and excitation voltage settings (Blue)

  • Implement or create a saved sensor setup (Green)

  • Change the sample rate of specific channel (Orange)

  • Set precision of the channel (Pink)
